## Problem Statement
Design and implement a DL model capable of learning the underlying structure, grammar, and contextual dependencies of a given text corpus to generate coherent and meaningful text sequences using Vanilla RNN, LSTM, and GRU.

## Contents
- Tokenization and sequence generation from text corpus
- Model training: Vanilla RNN, LSTM, GRU
- Training loss comparison
- Text generation using trained models

## Requirements
```
tensorflow
numpy
matplotlib
```

## How to Run
1. Open `Week5_TextGeneration_RNN_LSTM_GRU_MaheshShinde.ipynb` in Jupyter or Google Colab
2. Run all cells top to bottom
3. View loss comparison graph and generated text outputs

## Conclusion
LSTM and GRU handle long-term dependencies better than Vanilla RNN, with GRU being computationally faster and giving comparable results.
