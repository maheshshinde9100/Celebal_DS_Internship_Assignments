# End-to-End Machine Learning Pipeline on Tesla Deliveries & Sales Data

## Submission Details

* **Assignment:** End-to-End Machine Learning Pipeline
* **Dataset:** Tesla EA Deliveries & Production Data (2015–2025)
* **Submitted By:** *Mahesh Shinde*
* **College:** *Sanjivani College of Engineering*

## Steps Performed

1. Loaded the Tesla dataset from Kaggle.
2. Performed data preprocessing:

   * Handled missing values
   * Removed outliers
   * Encoded categorical features
3. Conducted Exploratory Data Analysis (EDA) using visualizations.
4. Performed feature engineering by creating new meaningful features.
5. Split the data into training and testing sets.
6. Trained and evaluated multiple regression models:

   * Linear Regression
   * Ridge Regression
   * Lasso Regression
   * Random Forest
   * Gradient Boosting
   * XGBoost
   * LightGBM
7. Tuned the best-performing model using GridSearchCV.
8. Performed time series forecasting using:

   * Holt-Winters Exponential Smoothing
   * SARIMA
9. Generated a 12-month forecast for Tesla deliveries.
